# Privacy Policy

**PDF to Markdown Converter** does not collect, store, transmit, or share any personal data.

## Data Collection

This extension collects **zero data**. Specifically:

- **No personal information** — No name, email, IP address, or any identifier is collected, stored, or sent anywhere.
- **No usage analytics** — No page views, button clicks, or feature usage is tracked.
- **No cookies** — No cookies, localStorage, or any other persistent storage is used.
- **No network requests** — All PDF processing happens entirely on your device. No data ever leaves your browser.

## Permissions

This extension requests the following permission:

- **`downloads`** — Used solely to save the converted Markdown files to your computer. The download is triggered by you, and you control where the file is saved.

## How It Works

1. You select or drag-and-drop PDF files into the extension.
2. The PDF is parsed locally using [pdf.js](https://mozilla.github.io/pdf.js/) (Mozilla's PDF engine — the same one built into Firefox).
3. The content is converted to Markdown entirely within your browser.
4. The resulting `.md` file is saved to your Downloads folder.

No data is sent to any server at any stage. Everything stays on your machine.

## Third-Party Code

This extension bundles [pdf.js](https://mozilla.github.io/pdf.js/) and [@pdf2md/core](https://github.com/nickcoutsos/pdf2md). Both are open-source libraries that operate entirely client-side. No external services are contacted.

## Changes to This Policy

If this policy changes in the future, the update will be posted on this page and noted in the extension's version history.
